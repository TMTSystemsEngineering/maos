Detailed Description

The following are blas and lapack fortran function definitions. Notice that when generating MKL custom libraries, use lp64 instead of ilp64 for blas/lapack routines to be compatible when usual blas/lapack definitions.

2013-12-17: replaced int* by ptrdiff_t* to be compatible with blas/lapack that expect 64 bit integers. Still backward compatible with 32bit integers because only one number will be used and we are using little indians. 2014-03-19: replaced ptrdiff_t by long.
