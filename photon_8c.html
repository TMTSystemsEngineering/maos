Compute subaperture star flux from magnitude, subaperture size, sample period, etc.

Parameters
[in]zb[in] Sky background and zero magnitude flux
[out]Np[out] number of total signal at each wvl.
[out]Nptot[out] total signal (sum of Np).
[out]Nbtot[out] number of background photon per pixel
[out]QCSNR[out] signal to noise ratio for a Quadcell
[out]QCNEA[out] noise equivalent angle for a Quadcell with Nyquist sampling
[in]nwvl[in] number of wavelength.
[in]wvls[in] vector of wavelength
[in]mags[in] vector of magnitudes
[in]dxsa[in] subaperture side length
[in]iscircle[in] where the subaperture is circle/part of a circle. true for TT/TTFA sensors.
[in]pixtheta[in] pixel extense in radian
[in]dt[in] sampling period in seconds
[in]za[in] zenith angle
[in]strehl[in] Strehl of the image. set to 1 for full flux.
[in]imperrnm[in] Implementation error in nm
[in]thruput[in] end to end optical throughput
[in]qe[in] detector quantum efficiency.
[in]rne[in] detector read out noise.
