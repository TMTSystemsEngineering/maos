Detailed Description

Generate random stars from besancon star counting model. The data file contains J and H magnitudes of all stars within 5 square degree FoV. The location of the stars are assumed to be randomly distributed. Therefore, the possible number of stars within any sub-field is a poisson distribution with mean equal to the average density multiplied with area.
