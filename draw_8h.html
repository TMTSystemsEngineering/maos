void draw_helper ( void  )

A helper routine that forks in the early stage to launch drawdaemon.

We don't use the main routine to launch drawdaemon because it takes very long to fork a process that consumes Gig's of memory.

We don't use drawdaemon to launch it because drawdaemon may not have the right DISPLAY related environment and also hard to work during ssh session even if environment is passed.
