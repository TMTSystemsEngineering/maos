Detailed Description

Routines to read .conf type config files.

Each entry is composed of a key and a value. The entries are maintained a hash table. Each entry can be retrieved from the key.

Start the read config process by opening .conf files and fill the entries in a hash table.

A key can be protected. If a key is not protected, newer entries with the same key will override a previous entry.

Parameters:
