Detailed Description

Decomposes a matrix into sparse and low rank terms and do math operations.

The forward operations can be represented by a matrix M and two low rank terms as y = ( M - u*v' ) *x , or by a function Mfun and corresponding argument Mdata.

This can also be used to solve linear equations using muv_solve() or muv_bgs_solve().

void muv_direct_prep ( MUV_T *  A,
double  svd 
)

Cholesky factorization (svd=0) or svd (svd>0) on the sparse matrix and its low rank terms.

if svd is less than 1, it is also used as the threshold in SVD inversion.

When there are low rank terms, Up=M^{-1}*U, Vp=M^{-1}*[V*(I-Up'*V)^{-1}]

void muv_direct_diag_prep ( MUV_T *  A,
double  svd

Cholesky factorization (svd=0) or svd (svd>0) on each diagonal element of the block sparse matrix and its low rank terms.

if svd is less than 1, it is also used as the threshold in SVD inversion.

When there are low rank terms, Up=M, Vp=M\[V*(I-Up'*V)^{-1}] For each diagonal block.
