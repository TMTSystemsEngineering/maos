Detailed Description

contains input parameters for simulation, like loop gain, seeds, etc.

        servo coefficient for high order dm.  A is command. e is

error signal. at time step n, the command is updated by A(n)=A(n-1)*apdm(0)+A(n-2)*ap(1)+...+e(n-2)*ep
