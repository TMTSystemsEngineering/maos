Detailed Description

contains input parameters for wavefront reconstruction.

Field Documentation

int RECON_CFG_T::alg

algorithm for reconstruction.

0: MVR. 1: LSR. moved from sim.recon

int RECON_CFG_T::glao

whether we are in GLAO mode where all WFS in each powfs are averaged

int RECON_CFG_T::split

split reconstruction/tomography type.

  • 0: integrated tomography
  • 1: adhoc split tomography
  • 2: minimum variance split tomography (only valid if recon.alg=0)
int RECON_CFG_T::warm_restart

Warm restart in CG.

int RECON_CFG_T::mvm 

                 Use the various algorithms recon.alg to assemble a control

matrix to multiply to gradients to get DM commands.

If the algorithm needs PSOL gradient, we will have an auxillary matrix to multiply to the DM actuators and subtract from the result.

int RECON_CFG_T::sock

-1: handle locally.

otherwise, indicate the socket that handles it.
