Detailed Description

contains input parameters for each type of wfs (powfs).

Field Documentation

double* POWFS_CFG_T::wvl

list of wavelength

char* POWFS_CFG_T::piinfile

input averaged pixel intensities for matched filter.

NULL to disable

char* POWFS_CFG_T::sninfile

Speckle noisy input file.

NULL to disable. not used

double POWFS_CFG_T::hs

height of guide star

double POWFS_CFG_T::saat

subaperture area (normalized) threshold to drop subaperture.

double POWFS_CFG_T::safill2d

subaperture lenslet throughgput.

value is used to alter amplitude map

double POWFS_CFG_T::saspherical

Subaperture spherical aberration in nm RMS at best focus.

double POWFS_CFG_T::safocuspv

Subaperture focus error in nm p/v.

char* POWFS_CFG_T::neareconfile 

            prefix of file contains noise equivalent angle in

radian.

_wfs# is added when reading file.

double POWFS_CFG_T::nearecon

NEA used in reconstruction.

double POWFS_CFG_T::neasim

NEA used in simulation.

-1 to use nearecon

char* POWFS_CFG_T::neasimfile

read NEA used in simulation from file.

Defined at sim.dt sampling rate, in radian. neasim must be -1

double POWFS_CFG_T::neaspeckle

NEA caused by speckle noise.

Added to matched filter estimation of NEA due to photon and detector noise in physical optics mode for reconstructor

double POWFS_CFG_T::bkgrnd

background in electron per pixel per LGS frame

double POWFS_CFG_T::bkgrndc 

         How much of the background in bkgrnd can be calibrated

out.

depends on variability.

char* POWFS_CFG_T::bkgrndfn 

         file contains sky background/rayleigh scatter input for

each subaperture in each wfs.

char* POWFS_CFG_T::bkgrndfnc 

        How much of the background in bkgrndfn can be

calibrated out.

depends on variability.

double POWFS_CFG_T::rne

read out noise in electron per pixel per frame

double POWFS_CFG_T::pixblur

pixel bluring due to leakage.

relative to pixel size.

double POWFS_CFG_T::dx

sampling of opd points in each subaperture.

usually matches atmosphere sampling for LGS. may be coraser for NGS.

double POWFS_CFG_T::pixtheta 

         size of pixel pitch along x/y or azimuthal if radial

ccd.

Converted to radian from user input

double POWFS_CFG_T::radpixtheta

size of pixel pitch along radial direction.

-1 for square pixel

double POWFS_CFG_T::fieldstop

size of field stop in arcsec.

double POWFS_CFG_T::pixoffx

offset of image center from center of detector

double POWFS_CFG_T::pixoffy

see pixoffx

double POWFS_CFG_T::sigscale

scale the signal level for simulation.

double POWFS_CFG_T::siglev

signal level.

will be override by wfs.siglev is specified.

double* POWFS_CFG_T::wvlwts

weights for each wavelength.

can be overriden by wfs.wvlwts.

struct LLT_CFG_T* POWFS_CFG_T::llt

configuration for LLT

char* POWFS_CFG_T::fnllt

filename of LLT configuration.

empty means no llt.

int POWFS_CFG_T::trs

tip/tilt removal flag.

True for LGS, False for NGS

int POWFS_CFG_T::dfrs

differential focus removal flag.

True for LGS, False for NGS

int POWFS_CFG_T::lo

whether this is a low order wfs.

False for LGS, True for NGS

int POWFS_CFG_T::skip

skip in high order tomography, for split tomo (derived parameter)

int POWFS_CFG_T::psol

Compute pseudo open loop gradients (derived parameter)

int* POWFS_CFG_T::wfs

array of wfs belongs to this powfs

int* POWFS_CFG_T::wfsind

wfsind[iwfs] gives the index of the wfs in this powfs group

int POWFS_CFG_T::nwfs

number of wfs belonging to this powfs

int POWFS_CFG_T::nwfsr

number of wfs for reconstruction belonging to this powfs

int POWFS_CFG_T::neaphy

use nea from physical optical precomputation in geometric simulations.

int POWFS_CFG_T::phyusenea 

           force using supplied noise equivalent angle in physical

optics simulations

int POWFS_CFG_T::order

order of wavefront sensing along one dimension.

int POWFS_CFG_T::pixpsa

number of detector pixels along x/y or azimuthal if radial CCD.

int POWFS_CFG_T::radpix

number of detector pixels along radial direction if radial CCD

int POWFS_CFG_T::radrot

For radial format CCD, rotate OTF into coordinate plane.

uses less memory

int POWFS_CFG_T::ncomp

number of PSF points before forming detector image.

0 for automatic

int POWFS_CFG_T::embfac

Embed subaperture atm OPD before fft.

set to 2.

int POWFS_CFG_T::nwvl

Number of wavelength.

1 for LGS, 2 for NGS J+H sensing.

int POWFS_CFG_T::gtype_sim

wfs type if not using physical optics in reconstruction.

  • 0: geometric
  • 1: ztilt.
int POWFS_CFG_T::gtype_recon

wfs type if not using physical optics in simulation.

  • 0: geometric
  • 1: ztilt.
int POWFS_CFG_T::phytype

physical optics type.

1: mtch, 2: tcog, 3: MAP

int POWFS_CFG_T::phytypesim

physical optics type for simulation.

-1 to follow phytype

int POWFS_CFG_T::phystep

frames to start using physical optics.

  • 0: means from frame 0.
  • >0: need to compute GS0 to calculate geometric optics
  • -1: never, doesn't need to compute DTF
int POWFS_CFG_T::usephy 

              whether physical optics is used at all during

simulation.

(derived parameter)

double POWFS_CFG_T::mtchcr

if >0 use constrained mtch for this amount of pixels

double POWFS_CFG_T::mtchcra

if >0 use constrained mtch for azimuthal for this amount of pixels

int POWFS_CFG_T::mtchcpl

use coupling between r/a measure error.

useful for LGS with x-y ccd.

int POWFS_CFG_T::mtchstc

shift peak in the time averaged short exposure PSF to center using fft.

int POWFS_CFG_T::mtchscl

scale subaperture image to have the same intensity as i0.

Keep false.

int POWFS_CFG_T::mtchadp

Using adaptive matched filter.

When the number of pixels in the image brighter than half maximum is more than this value, use constraint. introduced on 2011-02-21.

double POWFS_CFG_T::cogthres

CoG threshold, relative to max(im)

double POWFS_CFG_T::cogoff

CoG offset to remove, relative to max(im).

int POWFS_CFG_T::needGS0

need to compute GS0 (derived parameter)

int POWFS_CFG_T::noisy

noisy or not during simulation

int POWFS_CFG_T::ncpa_method

Method to correct ncpa.

  • 0: do nothing.
  • 1: apply gradient electronic offset.
  • 2: apply ncpa to average pixel intensity i0, better than 1
int POWFS_CFG_T::pistatout

output time averaged short exposure image.

1: shift to center, 2: do not shift

int POWFS_CFG_T::pistatstart

time step to compute pistatout

int POWFS_CFG_T::pistatstc

1: shift to center using fft method.

0: use geometric gradients.

int POWFS_CFG_T::psfout

output time history of low order wfs PSF.

never do this for LGS.

int POWFS_CFG_T::dtrat

ratio of sample period over fast loop (LGS)

int POWFS_CFG_T::idtrat

Index of dtrat into parms->sim.dtrats.

int POWFS_CFG_T::i0scale

scale i0 to matched subaperture area.

int* POWFS_CFG_T::scalegroup

scale group for dm propergation cache.

(derived parameter)

int POWFS_CFG_T::moao

index into MOAO struct.

-1: no moao

int POWFS_CFG_T::dither

Turn on/off dithering.

double POWFS_CFG_T::dither_amp

Dithering amptlidue to update centroid gain or matched filter.

double POWFS_CFG_T::dither_gpll

Gain of phase locked loop.

int POWFS_CFG_T::dither_nskip

Skip steps for uplink loop to stable.

int POWFS_CFG_T::dither_npll

Period for updating PLL.

int POWFS_CFG_T::dither_nstat

Period for updating statistics.
