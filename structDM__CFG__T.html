char* DM_CFG_T::misreg

misregistration of the DM with respect to the pupil, described by file containing a cell array A of 2x1.

The misregistered coordinate is computed as

xm(ip)={ic}(A{1,ip}(1,ic)*pow(x,A{1,ip}(2,ic))*pow(y,A{1,ip}(3,ic))) ym(ip)={ic}(A{2,ip}(1,ic)*pow(x,A{1,ip}(2,ic))*pow(y,A{1,ip}(3,ic))) where ip is ipowfs. ic is index of column in entries of A.

char* DM_CFG_T::hyst

File containing a matrix that describes the hysterisis.

The matrix should have 3 rows, and multiple columns. Each column describe the parameters for a mode, and the rows are: 1) weight 2) alpha, 3) beta. See "Hysteresis Modeling and Simulation" by Luc Gilles
